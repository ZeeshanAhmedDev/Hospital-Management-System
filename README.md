
# Hospital Management System
This project is a hospital management system built with Bootstrap, JavaScript, and Firebase.

## Features
- Patient and staff dashboards
- Authentication and user management
- Appointment scheduling

## Setup
1. Clone the repository.
2. Install dependencies: `npm install` (if required).
3. Set up Firebase and replace configuration in `src/firebase.js`.

## Run
Serve the `public` folder using any static file server or Firebase Hosting.
